# Spanish-English Translation Transformer

This repository contains a PyTorch implementation of a simple encoder-decoder Transformer model trained to translate English sentences to Spanish. It was developed as part of an assignment for a machine learning / NLP course. The notebook demonstrates data preparation, custom attention modules, training, evaluation, and inference.

## 📁 Repository Structure

- `M25AI2024_Assignment_2.ipynb` - Jupyter notebook with all code for data loading, preprocessing, model definition, training loop, and evaluation.
- `eng.*` / `spa.*` - SentencePiece tokenizer files generated during preprocessing (vocab and model files).
- `eng.txt` / `spa.txt` - Temporary plaintext files used to train tokenizers (generated by notebook).

> Note: the tokenizer files and text files are generated when the notebook runs and can be deleted or regenerated.

## 🛠️ Requirements

- Python 3.8 or higher
- PyTorch
- `datasets` (Hugging Face)
- `sentencepiece` (version 0.1.99)
- `torchtext`
- `matplotlib` for plotting
- `numpy`, `tqdm`

Install dependencies using pip:

```bash
pip install torch datasets sentencepiece==0.1.99 torchtext matplotlib numpy tqdm
```

## 🚀 How to Run

1. Open `M25AI2024_Assignment_2.ipynb` in JupyterLab or VS Code.
2. Execute the cells sequentially. They perform:
   1. Data loading from the `OscarNav/spa-eng` dataset (50k samples subset).
   2. Text cleaning and tokenization using SentencePiece (vocab size 7687).
   3. Definition of custom `MultiHeadAttention`, `FeedForward`, encoder/decoder layers, and full Transformer model.
   4. Training loop for 20 epochs with train/validation splits.
   5. Plotting of loss curves and greedy inference examples.

> GPU acceleration will be used if CUDA is available.

## 🔍 Model Details

- Vocabulary size: 7687 tokens for both English and Spanish.
- Embedding dimension: 256
- Attention heads: 4
- Transformer has a single encoder and decoder layer.
- Maximum sequence length: 20
- Loss: `CrossEntropyLoss` (ignoring padding index).

## 📈 Results

After training for 20 epochs, the notebook prints training and validation loss for each epoch and displays sample translations. Use the provided `translate()` function to test custom English sentences.
